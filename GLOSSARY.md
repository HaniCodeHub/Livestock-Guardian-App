# Glossary of Terms

## Technical Terms

### AI/ML Terms

**Biometrics**: Unique biological or behavioral characteristics used to identify individuals or animals. In Livestock Guardian, we use muzzle biometrics.

**Confidence Score**: A numerical value (0-1) indicating the certainty of an AI prediction. Higher scores mean more confidence.

**Feature Vector**: Mathematical representation of an image used by AI models for comparison and matching.

**Model Training**: Process of feeding data to an AI model to improve its accuracy.

**Overfitting**: When an AI model performs well on training data but poorly on new data.

**Transfer Learning**: Using a pre-trained model as a starting point for a new task.

### Database Terms

**Index**: A data structure that improves query speed by organizing data for faster lookup.

**Normalization**: Process of organizing database tables to reduce redundancy and improve data integrity.

**Primary Key**: Unique identifier for each record in a table.

**Foreign Key**: A field that links to the primary key of another table.

**Replication**: Creating copies of data across multiple servers for backup and availability.

### API Terms

**REST**: Representational State Transfer - architectural style for web services.

**Endpoint**: A URL that receives requests and returns responses.

**Payload**: Data sent in an API request.

**Query Parameter**: Additional information in a URL used to filter or modify results.

**Rate Limiting**: Restricting the number of requests a user can make in a time period.

**Webhook**: A way for applications to send real-time data to each other.

---

## Business Terms

**Farmer/Livestock Owner**: A person who owns and manages animals.

**Livestock**: Farm animals raised for agricultural purposes (cattle, sheep, goats, etc.).

**Ownership Verification**: Process of proving that a person owns a specific animal.

**Theft Report**: A report filed when an animal is missing or stolen.

**Premium Plan**: A paid subscription offering advanced features.

**Free Tier**: Basic free features available to all users.

---

## Product Terms

**Muzzle Pattern**: Unique pattern on an animal's muzzle used for identification.

**Biometric Registration**: Process of capturing and storing an animal's unique biological characteristics.

**Identification**: Process of determining which animal is in a photo.

**Dashboard**: User interface showing animal inventory and statistics.

**Mobile App**: Livestock Guardian application for Android devices.

**Web Portal**: Browser-based interface for accessing Livestock Guardian.

---

## Security Terms

**Encryption**: Converting data into code to prevent unauthorized access.

**Authentication**: Process of verifying a user's identity.

**Authorization**: Process of determining what a user is allowed to do.

**JWT (JSON Web Token)**: A secure method of transmitting information between parties.

**OAuth**: A standard protocol for secure authentication.

**Two-Factor Authentication (2FA)**: A security method requiring two forms of verification.

**Audit Log**: Record of all system activities for security and compliance.

---

## Infrastructure Terms

**Cloud**: Remote servers accessed via the internet.

**Server**: A computer that provides services to other computers.

**Database**: Organized collection of data stored on a server.

**API Gateway**: A server that manages API requests and responses.

**Load Balancer**: Distributes traffic across multiple servers for better performance.

**Cache**: Fast storage that reduces the need to access slower databases.

**CDN (Content Delivery Network)**: Network of servers that deliver content faster to users.

---

## Performance Terms

**Latency**: Time delay between request and response.

**Throughput**: Amount of data processed in a given time.

**Uptime**: Percentage of time a system is operational.

**Response Time**: Time taken by a system to respond to a request.

**Scalability**: Ability of a system to handle increased load.

**Concurrency**: Ability to handle multiple operations simultaneously.

---

## Data Terms

**Data Privacy**: Protection of personal information from unauthorized access.

**GDPR**: General Data Protection Regulation - EU data protection law.

**Data Breach**: Unauthorized access to protected information.

**Data Retention**: How long data is stored before deletion.

**Data Export**: Extracting data in a specific format.

**Backup**: Copy of data for recovery purposes.

---

## Development Terms

**Repository**: Central location where code is stored.

**Commit**: Saving changes to the code repository.

**Branch**: Separate version of code for development.

**Pull Request**: Proposal to merge code changes.

**Merge Conflict**: When changes to the same code cannot be automatically combined.

**Deployment**: Process of releasing code to production.

**CI/CD**: Continuous Integration/Continuous Deployment - automated testing and release process.

**Dependency**: External code library required by an application.

---

## QA Terms

**Test Case**: Specific scenario used to verify functionality.

**Unit Test**: Testing of individual components in isolation.

**Integration Test**: Testing of multiple components working together.

**Regression Test**: Testing that previous functionality still works.

**Bug**: A defect in the code causing unexpected behavior.

**Patch**: A small fix released to address a specific issue.

---

## Analytics Terms

**User Retention**: Percentage of users who continue using the service.

**Churn Rate**: Percentage of users who stop using the service.

**Active User**: A user who has used the service recently.

**Conversion Rate**: Percentage of users who complete a desired action.

**Engagement**: Level of user interaction with the application.

---

## Livestock Terms

**Species**: Type of animal (cattle, sheep, goats, etc.).

**Breed**: Specific type within a species.

**Microchip**: Electronic implant used for animal identification.

**Ear Tag**: Physical tag attached to an animal's ear for identification.

**Branding**: Marking an animal with a hot iron for identification.

**Herd**: Group of animals managed together.

**Veterinary**: Related to animal health and medicine.
