# Mobile App User Guide

## Getting Started

### Installation

#### From Google Play Store
1. Open Google Play Store
2. Search for "Livestock Guardian"
3. Tap **Install**
4. Grant required permissions:
   - Camera (for muzzle photos)
   - Location (for animal tracking)
   - Storage (for image storage)

#### From APK File
1. Download APK from [livestock-guardian.com/downloads](https://livestock-guardian.com/downloads)
2. Open file manager
3. Tap the downloaded APK file
4. Tap **Install**
5. Open the app from App Drawer

### First Launch

**Welcome Screen**:
- Read the introduction
- Learn about features
- Agree to terms and privacy policy
- Tap **Get Started**

**Create Account** or **Sign In**:
- **New User**: Tap **Create Account**
- **Existing User**: Tap **Sign In**

## Account Setup

### Creating an Account

1. **Email**: Enter your email address
2. **Password**: Create a strong password (12+ characters)
3. **Full Name**: Your name
4. **Phone Number** (optional): For notifications
5. **Location**: Your farm location (for tracking)
6. **Profile Photo** (optional): Take or upload a photo

Tap **Create Account** and verify your email.

### Account Security

**Recommendations**:
- Use a strong, unique password
- Enable Two-Factor Authentication (2FA)
- Keep your email address updated
- Review login activity in settings

**Enable 2FA**:
1. Go to **Settings** → **Security**
2. Tap **Enable Two-Factor Authentication**
3. Choose SMS or Authenticator App
4. Follow setup instructions
5. Save recovery codes in a safe place

## Registering Your First Animal

### Step 1: Open Registration
1. Tap the **+** (plus) button at bottom right
2. Select **Register New Animal**

### Step 2: Enter Details
1. **Animal Name**: Give your animal a name
2. **Species**: Select from dropdown (Cattle, Sheep, Goat, Horse, etc.)
3. **Breed**: Choose breed (optional but helpful)
4. **Birth Date**: Enter or estimate (optional)
5. **Gender**: Male/Female (optional)
6. **Color/Description**: Note distinctive features
7. **Microchip ID**: If available (optional)
8. **Ear Tag**: If available (optional)

### Step 3: Capture Muzzle Photo
1. Tap **Capture Muzzle Photo**
2. Position animal's muzzle in frame
3. Ensure good lighting
4. Avoid shadows on muzzle
5. Tap **Capture**
6. Review photo quality
7. Retake if needed, or tap **Use This Photo**

### Step 4: Review & Confirm
- Review all entered information
- Tap **Register Animal**
- Animal is now registered! ✓

**Tips for Better Results**:
- Use natural lighting (outdoor is best)
- Get close to the muzzle area
- Avoid direct shadows
- Take photos from slightly above
- Ensure muzzle is clear and visible

## Identifying Animals

### Quick Identification

1. Tap the **Camera** icon
2. Point camera at animal's muzzle
3. Tap **Capture** to take photo
4. App automatically identifies the animal
5. View results:
   - Matched animal name
   - Confidence score (0-100%)
   - Match details
   - Owner information

### Manual Search

1. Tap **Search** icon
2. Choose search type:
   - **By Name**: Enter animal's name
   - **By Microchip**: Enter ID
   - **By Ear Tag**: Enter tag number
3. View search results
4. Tap animal to view details

## Theft Reporting

### Report Missing Animal

1. Open animal profile
2. Tap **Menu** (three dots)
3. Select **Report Missing/Stolen**
4. Enter details:
   - Date last seen
   - Location
   - Description of circumstances
   - Any notable changes
5. Optionally:
   - Attach additional photos
   - Enter police report number
   - Add contact information
6. Tap **Submit Report**

**What Happens**:
- Alert sent to nearby app users
- Notification to authorities (where available)
- Animal flagged in system
- Can share on social media

### Resolve Theft Report

1. Open animal profile
2. Tap **Resolve Report**
3. Enter resolution status:
   - Animal found
   - Animal recovered
   - Still missing
   - Other
4. Add notes if desired
5. Tap **Update Status**

## Ownership Verification

### Generate Certificate

1. Open animal profile
2. Tap **Menu** (three dots)
3. Select **Generate Ownership Certificate**
4. Review certificate preview
5. Tap **Download PDF** or **Share**

### Certificate Contains
- Animal photo and details
- Owner information
- Biometric data reference
- Issue date and digital signature
- QR code for verification

### Share Certificate

1. Open animal profile
2. Tap **Share** button
3. Choose method:
   - **Email**: Send to buyer/authority
   - **WhatsApp**: Share via messaging
   - **Print**: Print physical copy
   - **Download**: Save to device

## Settings & Preferences

### General Settings
1. Tap **Menu** (hamburger icon)
2. Select **Settings**
3. Configure:
   - **Language**: App language
   - **Theme**: Light/Dark mode
   - **Notifications**: Enable/disable alerts

### Location Services

**Enable Location Tracking**:
1. Go to **Settings** → **Location**
2. Toggle **Location Services** ON
3. Grant permission when prompted
4. Animal locations tracked when app open

### Privacy Settings

1. Go to **Settings** → **Privacy**
2. Control:
   - **Profile Visibility**: Public/Private
   - **Animal Visibility**: Who can see your animals
   - **Data Sharing**: Research participation
   - **Location Privacy**: Share location details

### Notification Preferences

1. Go to **Settings** → **Notifications**
2. Toggle ON/OFF:
   - Identification results
   - Theft alerts
   - Account activity
   - System updates
   - New features

## Backing Up Data

### Automatic Backup
- App automatically syncs to cloud
- Happens when online
- No manual action needed
- All animals backed up

### Manual Backup

1. Go to **Settings** → **Backup**
2. Tap **Backup Now**
3. Wait for completion
4. Confirmation message appears

### Export Data

1. Go to **Settings** → **Data Export**
2. Choose format:
   - **CSV**: Spreadsheet format
   - **PDF**: Document format
   - **JSON**: Technical format
3. Tap **Export**
4. File saved or can email

## Troubleshooting

### App Won't Open
- Restart device
- Update app from Play Store
- Clear app cache (Settings → Apps → Livestock Guardian → Clear Cache)
- Reinstall app if needed

### Camera Not Working
- Grant camera permission (Settings → Apps → Permissions → Camera)
- Restart app
- Check if another app is using camera
- Restart device

### Can't Sign In
- Verify email address
- Reset password if forgotten
- Check internet connection
- Clear app cache

### Photos Not Uploading
- Check internet connection (WiFi recommended)
- Verify file size (< 10MB)
- Try again in a few moments
- Contact support if persistent

## FAQs

**Q: Can I use the app offline?**
A: Limited features work offline. Full sync happens when reconnected.

**Q: How accurate is identification?**
A: 95%+ accuracy with good quality photos in normal conditions.

**Q: Who can see my animal data?**
A: Only you by default. Adjust privacy settings to share selectively.

**Q: Is my data encrypted?**
A: Yes, all data encrypted in transit and at rest.

**Q: How often should I update photos?**
A: Update annually or after significant changes to animal.
